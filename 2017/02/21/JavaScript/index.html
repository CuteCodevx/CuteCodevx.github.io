JavaScript

JavaScript操作DOM常用API总结

创建型API

createElement,createTextNode,cloneNode,createDocumentFragment
注意：1.他们创建的节点都是孤立的节点，需要appendChild添加到文档中
     2.cloneNode要注意如果被复制的节点是否包含字节点及事件绑定等问题
        复制节点必须要修改id；
        如果是通过addEventListener或onclick进行绑定事件，则副本节点不会绑定，如果使用内联方式绑定，则副本节点同样会触发该事件
     3.使用createDocumentFragment来解决添加大量节点时的性能问题
        创建了一个fragment，每次生成的li节点先添加到fragment，最后一次性添加到list。

页面修改型API

appendChild,insertBefore,removeChild,replaceChild
注意：removeChild->删除指定的节点并返回；

节点查询型API

document.getElementById,document.getElementsByTagName,document.getElementsByName,
document.getElementsByClassName,document.querySelector,document.querySelectorAll

节点关系型API

parentNode/parentElement
previousSibling/previousElementSibling/nextSibling/nextElementSibling
childNodes/children/firstNode/lastNode/hasChildNodes

元素属性型API

setAttribute
getAttribute

元素样式型API

window.getComputedStyle
getBoundingClientRect
