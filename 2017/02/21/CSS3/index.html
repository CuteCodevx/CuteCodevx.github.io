结构选择器

属性选择器

伪类选择器

E:nth-child(n) 从E标签的父级去找第n个标签，并且这个标签必须为E，否则找不到(按顺序找)
E:nth-last-child(n) 倒着找
E:nth-child(odd) 奇数行
E:nth-child(even) 偶数行
E:nth-child(1n) 1的倍数
E:nth-of-type(n) 从E标签的父级去找第n个E标签
E:nth-last-of-type(n) 倒着找

target伪类

E:target 修改锚点
div{display:none}
div:target{display:block}
<a href='#div1'></a><div id="div1"><div> 可以实现类似选项卡

checked伪类

E:checked checkbox/radio被选中后的状态 E->input[type=’radio/checkbox’]

文本伪类

E:fitst-line E标签里第一行文字的状态
E:fitst-letter E标签里第一个文字的状态
E::selection E标签里文字被选中的状态
E::before 在E标签的最前面添加内容
E::after 在E标签的最后面添加内容

not伪类

E:not(s) 排除掉某一个元素 s->要排除掉的那个标签的id或class
E~F 找到E标签后面的所有F标签

参数2 Y轴的偏移量

参数4 阴影的颜色

文字描边

-webkit-text-stroke webkit内核专有
参数1 描边的大小

盒模型阴影

box-shadow
参数1(可选)：阴影的方向 inset 内阴影，省略 外阴影
参数2 X轴的偏移量
参数3 Y轴的偏移量
参数4(可选) 阴影的扩展半径
参数5 阴影的大小
参数6 阴影的颜色

圆角

border-radius

transition

参数1 过渡的时间
参数2 过渡的属性 all->所有的属性 默认
参数3 过渡的方式 ease ->减速(默认)
linear ->匀速
ease-in ->加速
ease-out ->减速
ease-in-out ->先加速再减速
cubic-bezier ->贝塞尔曲线 有专门网站生成
参数4 delay

transform

只对块元素有作用
rotate(0deg) 旋转
skew(0deg) 倾斜
scale(倍数) 倍数
translate(像素) 位移 左上为－ 右上为+
先写的后执行，后写的先执行
transform-origin 变化的基点
transform-origin:left/right/center top/center/bottom;
让父级变成3d空间 transform-style:preserve-3d;perspective(800px)->井深

animation

参数1：运动的时间
参数2：关键帧的名字
参数3：过渡的方式，ease是默认的（减速）
linear 匀速
ease-in 加速
ease-out 减速
ease-in-out 先加速后减速
cubic-bezier 贝塞尔曲线
参数4：重复的次数
数字
infinite 无限次
参数5：动画的顺序
alternate 先正再返
normal 正常顺序
animation-play-state 播放与暂停
：paused
：running

Flex(弹性)布局

容器的属性(父级)

flex-direction

项目的排列方向，主轴的方向
4个值：row         (默认)主轴为水平方向，从左到右 
      row-reverse      从右到左 
      column           主轴为垂直方向，从上到下
      column-reverse   从下到上

flex-wrap

3个值：nowrap      (默认)不换行，宽度若超过父级则自动缩小
      wrap              换行
      wrap-reverse      换行后原先的第一行在下方

flex-flow

flex-direction与flex-wrap的简写：默认值为 row nowrap；

justify-content

项目在主轴上的对齐方向
5个值：flex-start (默认)左对齐
      flex-end        右对齐
      center          居中 
      space-between   两端对齐，中间间隔相等
      space-around      每个项目两侧间隔相等

align-items

垂直对齐
5个值：flex-start    交叉轴起点对齐
      flex-end      交叉轴终点对齐
      center        中点对齐
      stretch       如果项目未设置高度，将占满整个容器的高度
      baseline      项目的第一行文字的基线(文字底部)对齐

align-content

多行对齐
6个值：flex-start
      flex-end
      center
      space-between;
      space-around
      stretch         (默认)

项目的属性(子元素)

order

定义项目的排列顺序，数值越小，排列越靠前，默认为0

.item{order:<integer>;}

flex-grow

定义项目的放大比例，默认为0

.item{flex-grow:<number>;}

flex-shrink

定义项目的缩小比例，默认为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。

flex-basis

在分配多余空间之前，项目占据的主轴空间，默认值为auto，即项目的本来大小

flex

是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto
该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。
后两个属性值可选。

align-self

允许单个项目有与其他项目不一样的对齐方式，默认auto
