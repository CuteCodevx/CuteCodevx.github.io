如何实现响应式布局

CSS3-Media Query

常见属性：device-width，device-height --屏幕宽高
width，height  --可视区
orientation    --设备方向
resolution     --设备分辨率
使用:外联：<link rel="stylesheet" href="link.css" media="only screen and (max-width:480px)"/>
    内联：<style>
            @media screen and (min-width:480px){
                样式
            }
         </style>

Bootstrap

<meta name="viewport" content="width=device-width,initial-scale=1.0">
visible-xs 在极小的情况下展示
