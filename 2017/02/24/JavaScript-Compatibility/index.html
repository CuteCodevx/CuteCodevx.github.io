JavaScript Compatibility

浏览器兼容性

1.getElementsByClassName          IE9以及以上支持
2.querySelector/querySelectorAll  IE8以及以上支持
3.getComputedStyle(obj)[attr]       IE9以及以上支持  IE的属性 currentStyle 与之类似
4.trim()                        IE9以及以上支持      用正则表达式封装trim  return this.replace('/^\s+|\s+$/g','')
5.foreach()                    IE9以及以上支持  foreach(function(value,index,array){})
6.firstChild,lastChild,nextSibling,previousSibling   
    方法兼容，但取到的结果不一样，标准浏览器下firstChild.nodeName可能是#text文本节点，非标准浏览器取到的是标签
7.firstElementChild,lastElementChild,nextElementSibling,previousElementSibling
    IE9以及以上支持
8.window.pageXOffset
  window.pageYOffset  IE9以及以上支持
9.scrollTop 
    document.documentElement.scrollTop 其他的浏览器从document身上去获取
    document.body.scrollTop             谷歌从body身上获取
10.事件对象    IE9以及以上支持  IE678是window身上的属性(window.event)
11.addEventListener   IE9以及以上支持 
    IE678支持 attachEvent   但是它的事件名字要加on
              detachEvent  
function addEvent(obj,type,fn){
    if(obj.addEventListener){
        obj.addEventListener(type,fn)
    }else{
        obj.attachEvent(on+'type',fn);
    }
};
addEvent(box,'click',function(){
    alert(1);
})
