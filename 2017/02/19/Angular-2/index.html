Angular 2

Angular 2 组成

-模块：Modules，
    ES6中，提供import来导入在其他文件中定义的模块，用export导出如Jquery这样的依赖到业务代码模块中
-指令：Directives，
    指令可以为指定DOM元素添加新的行为特征，扩展元素功能。有可自定义特性。
-组件：Components，
    处理界面和交互相关的。
    复用。组件树。变化检测是Angular在应用的数据变化后，用于决定哪个组件需要随之刷新的机制
-模板：Templates，
    当使用组件标签时，通过template或temolateUrl属性引入HTML来描述渲染的界面内容。
-数据绑定：Data Binding，
    把数据映射到模板上，或从模板中取回数据 
-服务：Services
    书写和放置可重用的公共功能和复杂业务逻辑。
    一个服务就是一个简单的类，通常在组件中引用服务来处理数据和实现逻辑。    
-依赖注入：Dependency Injection
    通过对实现服务的类加上@Injectable装饰器，同时把他注册到Provider，从而将服务提供给调用者使用

开发语言 TypeScript

变量类型

boolean，number，string，array，null和undefined，

元组类型 tuple，

用来表示已知元素数量和类型的数组
let x:[string,number]; x=['Angular',25]; x=[25,'Angular'] //报错 console.log(x[0]) //Angular

枚举类型 enum，

可以手动修改默认下标值
enum Color {Red,Green,Blue}; let c:Color=Color.Blue; console.log(c) //2

enum Color {Redreen=6}; let c:Color=Color.Blue; console.log(c); //3

任意值类型 any,
void类型，

一个函数没有返回值时，意味着返回类型是void

never类型

声明为never类型的变量只能被never类型所赋值

声明变量时需要加上类型声明
let words:string=模板字符串反引号 你好，今年是${name}发布1周年 模板字符串反引号；
let flag:boolean=true;

生命周期支持

一些例子：
constructor 构造器初始化
OnChanges 接收来自父组件传入的数据，为接下来组件初始化提供数据持
OnInit 组件初始化
OnChanges 稳定期，数据变化就会触发
OnDestroy 组件销毁前

装饰器

赋予一个类更丰富的信息（元数据）

数据绑定

属性绑定 把组件类的数据传递到模板中 [value]
事件绑定 把模板里产生的数据通过函数调用的方式传递到组件类
双向绑定 [(ngModel)]

@Input 子组件的输入接口，用来接收来自父组件的数据 属性绑定 自上而下
@Output 子组件往父组件传递数据 事件绑定 自下而上
Output与EventEmitter配合使用

指令

属性指令
结构指令 改变模板的结构 ngIf用来插入或移除DOM节点
指定需要导入ElementRef,Renderer来辅助元素的渲染，导入后还要在构造器函数里添加对应的参数constructor(el:ElementRef,render:Renderer)

服务

依赖注入配置：providers：[xxxxx] 关键步骤 依赖注入机制会根据providers提供的xxxxx类型预先实例化这个对象并缓存到注入器里，组件构造函数里有constructor(xx:xxxxx)指定xx为xxxxx类型
分层注入：组件树里的不同层级，重新注入，不会影响其他
